# Audio Files

Place your audio files (.mp3, .wav, .ogg) in this folder.

The website will randomly select and autoplay one audio track on each page load.

## Supported formats:
- MP3 (.mp3)
- WAV (.wav)
- OGG (.ogg)

## Example files to add:
- background-1.mp3
- background-2.mp3
- background-3.mp3
